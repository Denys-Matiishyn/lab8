# Auction REST API

Цей проєкт є частиною лабораторної роботи з дисципліни "Організація баз даних".
Він реалізує REST API для управління даними аукціону (зокрема, таблицею покупців).

## Технології
* Python 3
* Flask
* MySQL

## Як запустити
1. Встановіть залежності: `pip install -r requirements.txt`
2. Налаштуйте базу даних MySQL (див. `schema.sql`).
3. Запустіть сервер: `python app.py`
4. Запустіть тести: `python test_api.py`
